## Description

Returns the uppercase subject string.


## Arguments

### Required

| Name | Type | Description |
|------|------|-------------|
| `subject` | `string` | Subject string |


## Returns

**Type:** `string`

## Examples

```code
search(upper(@), "string") -> "STRING"
```
