## Description

Returns the lowercase subject string.


## Arguments

### Required

| Name | Type | Description |
|------|------|-------------|
| `subject` | `string` | Subject string |


## Returns

**Type:** `string`

## Examples

```code
search(lower(@), "STRING") -> "string"
```
